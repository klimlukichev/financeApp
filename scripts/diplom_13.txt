1.3 Выбор программных средств

Для разработки рекомендательной системы по подбору художественной литературы необходимо тщательно подойти к выбору языков программирования и сопутствующих технологий. Поскольку система предполагает сложную обработку данных (анализ текстов, работу с большими объемами информации), требуется язык, сочетающий производительность, богатый набор библиотек для анализа данных и удобство разработки. Для сравнительного анализа были выбраны три современных языка: Java, Python и C#.

Java

Достоинства:

высокая производительность благодаря компиляции в байт-код и оптимизациям JVM;

наличие мощных фреймворков для веб-разработки (Spring) и работы с большими данными (Apache Spark);

строгая типизация и развитые инструменты рефакторинга, упрощающие поддержку кода;

Недостатки:

платные лицензии для коммерческого использования некоторых версий.

Python

Достоинства:

обширная экосистема библиотек для машинного обучения (scikit-learn, TensorFlow, PyTorch) [2] и обработки естественного языка (NLTK, spaCy, TextBlob) [6];

простота и читаемость кода, что ускоряет разработку и упрощает поддержку системы;

наличие специализированных библиотек для работы с матрицами (NumPy) и разреженными данными (SciPy), что критично для алгоритмов рекомендательных систем [7];

кроссплатформенность и большое сообщество разработчиков.

Недостатки:

относительно низкая производительность в вычислительно сложных задачах по сравнению с компилируемыми языками;

динамическая типизация может усложнять отладку и поддержку крупных проектов.

C#

Достоинства:

высокая производительность, сравнимая с Java;

удобные инструменты для разработки в экосистеме .NET (Visual Studio, LINQ);

Недостатки:

ориентация на Windows-платформу, хотя существуют и кроссплатформенные решения (.NET Core);

гораздо меньшее по сравнению с Python количество специализированных библиотек для анализа текстов;

является платным для коммерческого использования.

В результате сравнительного анализа языков были выбраны средства для создания программного обеспечения. Для компонента машинного обучения и обработки текстов оптимальным выбором является Python благодаря богатому набору специализированных библиотек и простоте реализации алгоритмов [3, 16]. В крупных проектах для веб-интерфейса и серверной части логичнее использовать Java (Spring Framework) или C# (ASP.NET Core), так как они обеспечивают лучшую производительность и безопасность для production-решений. Однако для задач выпускной квалификационной работы, где важны скорость разработки и доступность специализированных библиотек, более целесообразно использовать Python в сочетании с Flask.

В итоге был выбран следующий стек технологий:

основной язык разработки: Python (для алгоритмов рекомендаций и анализа текстов);

веб-фреймворк: Flask (для создания API и backend-логики);

база данных: PostgreSQL (для хранения информации о книгах и пользователях);

дополнительные библиотеки: scikit-learn (реализация SVD и других алгоритмов), TextBlob (анализ тональности текстов), SQLAlchemy (работа с базой данных).
